HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The information in the head element is useful for the browser and SEO
  • Child elements are nested inside of a parent element (e.g. body)
  • An element refers to the entire element from the opening tag to the closing tag
  • The closing tag always has a "/" to represent a closure.
  • Within the "img" element, the src attribute is the most important because it defines the location of the image
  • The "section" elements are semantic elements that define a section in a document that contains thematic content
  • The "p" element represents a paragraph or a block of text
  • The "ul,ol, and li" elements represent unordered lists, ordered lists, and list items
  • The "br" element creates a line of empty space, or a line break, between two blocks of content
  • The "a" element (which stands for anchor) creates links to the same webpage or other webpages
  • The "href" attribute points to the URL for the link

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Using external CSS style sheets is usually the preferred method as it allows developers to keep all their CSS rules in a seperate file, which makes design changes easier
  • A selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations contain two important components: the CSS property we want to apply and the value of the property
  • The special wild card symbol, "*", is used when applying a rule to all elements visible on the page
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
